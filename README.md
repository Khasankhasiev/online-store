# Online store, Sneakers

Интернет-магазин по продажам кроссовок

## Стэк технологий:

- Vue 3
- Vue Router
- Composition API
- TailwindCSS
- Axios
- @formkit/auto-animate
- Mokky
- ESLint
- Prettier

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
